> Fast and simple SSH library for interactive session based on paramiko.

## Supported Platforms
Currently tested on:
- Cisco switches (IOS,IOS-XE,Nexus)
- Cisco routers (IOS,IOS-XR,IOS-XE)
- HP (Model 5920AF)
- HUAWEI (NE40)

## Install

```sh
pip install pynet-ssh
```
